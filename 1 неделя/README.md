# Неделя 1 — Основы языка Go

## Задачи

### Понедельник — типы, переменные, fmt ✅
1. fmt.Println/Printf/Sprintf, базовые типы, zero values
2. Константы (const, iota)
3. Короткое объявление :=, var блоки
4. Конвертация типов
5. Мини-программа: структура данных учёта времени

### Вторник — функции, math, strconv
1. Функции: параметры, return
2. Multi-return (несколько возвращаемых значений)
3. Именованные возвраты
4. math пакет (Sqrt, Pow, Abs, Round)
5. strconv (Atoi, Itoa, ParseFloat)

### Среда — срезы, time
1. arrays
2. срезы (make/append/len/cap/copy)
3. time.Now/Since/Add/Unix
4. time.Sleep/Format
5. Работа со временем в задачах трекера

### Четверг — maps, structs
1. maps (make/delete/range)
2. structs: поля и конструкторы
3. Методы на структурах
4. value vs pointer receiver
5. Модель данных трекера времени

### Пятница — ошибки, defer
1. if/else/switch
2. for/range/break/continue
3. errors.New/fmt.Errorf
4. errors.Is/errors.As
5. defer, panic/recover

### Воскресенье — проект
CLI утилита учёта рабочего времени (старт/стоп/отчёт в файл)
